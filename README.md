# Code for Life

This project aims to introduce first year students in Biological Sciences at the University of Liverpool to statistics via three key tools.

  + Google Sheets
  + R
  + Python

## Contents

  + Descriptive Statistics
    - Central tendancies
    - Descrete data 
    - Continous data
      + Ordinal data 
      + Categorical data
  + Inferential Statistics
    - Regression
    - Sampling
    - Box plots
    - T-tests
      + Independent
      + Paired
    - Chi-squared
